Alert Notice Boxes allows you to create beautiful custom alerts that appear on pages or posts of your choice.

== Description ==

Alert Notice Boxes are no longer supported.
Our team has developed a better plugin with more options.
The plugin is called <a href="https://wordpress.org/plugins/advanced-notifications/"><strong>Advanced Notifications - click here</strong></a>

== Installation ==

Upload the Alert Notice Boxes plugin to your plugin folder on your site, Activate it.
